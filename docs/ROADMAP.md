# Vision

Да изградим модерно дигитално меню за Lokum, което да работи еднакво добре на:

- 🌐 Web
- 🤖 Android
- 🍎 iPhone (след версия 1.0)

Основни принципи:

- бързо
- красиво
- лесно за използване
- лесно за поддръжка
- лесно за разширяване
